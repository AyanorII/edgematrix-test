# Edgematrix assessment

App built using Vue.js and TypeScript for Edgematrix's test assessment, where the user can:

- See the profile of users
- Search by name
- Sort by name and bio length
- See details of a user

Link to project: https://edgematrix-test-4wvu-ayanorii.vercel.app/

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
